# Event Taxonomy (placeholder)

Allowed event types for placeholder ingestion:

- `gaze` (may include `x`, `y`, `viewport_width`, `viewport_height`, `confidence`, `timestamp`)
- `click`
- `scroll`
- `task_start`
- `task_complete`
- `calibration`
- `quality`
- `page_view`

Privacy rule: events must not contain raw media fields or webcam frame content.
