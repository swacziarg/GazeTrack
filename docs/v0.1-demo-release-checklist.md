## Privacy and safety

- [x] Synthetic mode does not require webcam permission.
- [x] Browser gaze experiment is feature-flagged, consent-gated, guarded against missing `window.webgazer`, and not bundled.
- [x] Docs state that raw webcam video, frames, images, screenshots, blobs, and base64 media must not be persisted or returned.
- [x] Backend rejects media-like payload keys before persistence.
- [x] Docs avoid medical-grade, biometric identity, perfect-accuracy, and production webcam tracking claims.
