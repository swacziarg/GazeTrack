# Release Readiness Checklist

- [ ] API contracts and sample payloads are up to date
- [ ] Privacy and safety docs reflect current behavior
- [ ] No raw webcam video handling introduced
- [ ] Non-goals and limitations are clearly documented
- [ ] Validation scripts/tests pass on clean checkout
- [ ] Changelog/README updated with current status
- [ ] Known risks and follow-up tasks are captured
- [ ] Rollback/mitigation notes prepared for risky changes
- [ ] PR checklist fully completed
- [ ] Reviewer sign-off received for privacy-sensitive changes
